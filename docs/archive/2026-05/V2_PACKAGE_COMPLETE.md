```bash
pg_dump lera > backup_v1_$(date +%Y%m%d).sql
```

#### Step 2: Create Staging Database
```bash
createdb lera_v2_staging
psql lera_v2_staging < backup_v1_20251220.sql
```

#### Step 3: Run Migration
```bash
psql lera_v2_staging < database/init/migration_v1_to_v2.sql
```

#### Step 4: Verify
```bash
psql lera_v2_staging -c "SELECT COUNT(*) FROM tenants;"
# Should return: 1 (LERA tenant)

psql lera_v2_staging -c "SELECT COUNT(*) FROM users WHERE tenant_id IS NOT NULL;"
# Should return: Same as total users
```
